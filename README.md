# 🎉 Magical Party - Márcia Gorete | Website para Festa de 15 Anos

Um sistema completo desenvolvido para a festa de 15 anos de **Márcia Gorete do Carmo Medeiros**, reunindo todas as informações do evento em uma plataforma moderna, elegante e responsiva.

O projeto oferece uma experiência interativa para os convidados, permitindo confirmação de presença, reserva de presentes, mural de mensagens, galeria de fotos, retrospectiva em vídeo e um painel administrativo completo para gerenciamento do evento.

## 🗄️ Banco de Dados

O projeto possui arquitetura híbrida.

Quando configurado:

- ✅ Supabase

Caso contrário:

- ✅ LocalStorage (modo offline)

Essa abordagem permite que o sistema continue funcionando mesmo sem conexão com o banco de dados.
